1/24 Note on 2.3, #6(a)

Part (a) is really separate from the rest of problem 6. In part (a), we're imagining that we drop an object from a height h, and we want to show that the object's velocity just before it hits the ground is the square root of 2gh.

To do that, we need a formula for the object's velocity, v(t), and we also need to know the time the object hits the ground so we can plug it into v(t). To find when the object hits the ground, it will be handy to have the formula for the object's position; let's call it y(t).

For motion problems like this, we start with the forces on the object, which create acceleration. In our problem, the only force is gravity, and it causes an acceleration of a(t) = -g.

Acceleration is the derivative of velocity, so to get v(t), we integrate: v(t) = ∫ a(t) = -gt + c. To find c, we use the fact that the object starts with zero velocity: v(0) = 0, so c = 0. Similarly, since velocity is the derivative of position, we can now integrate v(t) to get y(t): y(t) = ∫ v(t) = -gt2/2 + c. Now, since the object starts at height h, we get the initial condition y(0) = h, so c = h.

Now we have formulas for y(t) and v(t). To find when the object hits the ground, we can solve for when y(t) = 0. I'll let you finish it off from there.

1/15 Notes on Homework 1

I've had a few questions on the homework that I think could use some explanation, so here are some pointers:

2.1, #24(c):

The key thing to describe is the limit of the solution as t goes to zero, when a = 1/e. (In other words, find the value for your constant, c, that corresponds to the initial condition y(1) = a = 1/e, and plug it into the general solution, then take the limit as t tends to 0).

The reason a = 1/e is special is that for larger values of a, the solution y(t) tends to positive infinity as t -> 0, and for smaller values of a, it tends to negative infinity as t -> 0. Part (c) is asking you to investigate what happens right when a = 1/e.

2.2, #21:

You can leave your solution in implicit form (instead of solving for y, which would be a mess). As a rule, when solving separable equations, the solutions you get will be in implicit form, and it's OK to leave them that way, both on homework and on tests.
